# TCP vs UDP

TCP and UDP are communication protocols used for transmitting data across networks.

---

# TCP (Transmission Control Protocol)

TCP is connection-oriented and reliable.

It:
- checks for errors,
- guarantees delivery,
- and sends data in order.

## Common TCP Uses
- Web browsing (HTTPS)
- Email
- File transfers
- SSH connections

---

## TCP Characteristics

- Reliable
- Slower
- Error checking
- Ordered delivery

---

# UDP (User Datagram Protocol)

UDP is connectionless and faster than TCP.

It does not guarantee delivery or order.

## Common UDP Uses
- Video streaming
- Online gaming
- Voice calls
- DNS queries

---

## UDP Characteristics

- Faster
- Less reliable
- No error correction
- Lower overhead

---

# Comparison Table

| Feature | TCP | UDP |
|---|---|---|
| Reliability | High | Low |
| Speed | Slower | Faster |
| Error Checking | Yes | Limited |
| Connection Required | Yes | No |

---

# Why This Matters in Cybersecurity

Cybersecurity analysts monitor TCP and UDP traffic to:
- detect suspicious activity,
- identify attacks,
- and analyze network behavior.

Understanding these protocols is important for:
- packet analysis,
- firewalls,
- intrusion detection,
- and network security.
