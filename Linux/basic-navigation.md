# Basic Linux Navigation

## `pwd`

Shows the current working directory.

```bash
pwd
```

---

## `ls`

Lists files and folders.

```bash
ls
```

---

## `cd`

Changes directories.

```bash
cd Downloads
```

Go back one directory:

```bash
cd ..
```

Return to home directory:

```bash
cd ~
```

---

## `mkdir`

Creates a new directory.

```bash
mkdir cybersecurity
```

---

## `touch`

Creates a new file.

```bash
touch notes.txt
```

---

## `rm`

Deletes a file.

```bash
rm notes.txt
```

---

## `cp`

Copies files.

```bash
cp file1.txt backup.txt
```

---

## `mv`

Moves or renames files.

```bash
mv old.txt new.txt
```
