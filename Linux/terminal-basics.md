# Terminal Basics

The Linux terminal allows users to interact with the operating system using commands.

## Opening the Terminal

Most Linux distributions include a terminal application where commands can be entered.

Common terminal shortcuts:
- `Ctrl + C` → Stop a running command
- `Ctrl + L` → Clear the terminal screen
- `Tab` → Auto-complete commands or file names
- `Arrow Up` → Show previous commands

---

## Important Commands

### `clear`

Clears the terminal screen.

```bash
clear
```

---

### `echo`

Displays text in the terminal.

```bash
echo Hello World
```

---

### `history`

Displays previously used commands.

```bash
history
```

---

### `man`

Shows the manual page for a command.

```bash
man ls
```

---

### `whoami`

Displays the current logged-in user.

```bash
whoami
```

---

### `uname -a`

Displays detailed system information.

```bash
uname -a
```
